### `wezterm-attention`

You can import type annotations for
[`pro-vi/wezterm-attention`](https://github.com/pro-vi/wezterm-attention) as shown below:

```lua
---@type WeztermAttention
local attention = wezterm.plugin.require("https://github.com/pro-vi/wezterm-attention")
```
