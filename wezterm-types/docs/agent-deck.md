### `wezterm-agent-deck`

You can import type annotations for
[`Eric162/wezterm-agent-deck`](https://github.com/Eric162/wezterm-agent-deck) as shown below:

```lua
---@type AgentDeck
local agent_deck = wezterm.plugin.require("https://github.com/Eric162/wezterm-agent-deck")
```
