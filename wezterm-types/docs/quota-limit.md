### `wezterm-quota-limit`

You can import type annotations for
[`EdenGibson/wezterm-quota-limit`](https://github.com/EdenGibson/wezterm-quota-limit) as shown below:

```lua
---@type QuotaLimit
local ql = wezterm.plugin.require("https://github.com/EdenGibson/wezterm-quota-limit")
```
