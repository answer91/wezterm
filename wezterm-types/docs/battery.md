### `battery.wez`

You can import type annotations for
[`rootiest/battery.wez`](https://github.com/rootiest/battery.wez) as shown below:

```lua
---@type BatteryWez
local battery = wezterm.plugin.require("https://github.com/rootiest/battery.wez")
```
