### `resurrect.wezterm`

[`MLFlexer/resurrect.wezterm`](https://github.com/MLFlexer/resurrect.wezterm) as shown below:

```lua
---@type Resurrect
local resurrect = wezterm.plugin.require("https://github.com/MLFlexer/resurrect.wezterm")
```
