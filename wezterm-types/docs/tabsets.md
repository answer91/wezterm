### `tabsets.wezterm`

You can import type annotations for
[`srackham/tabsets.wezterm`](https://github.com/srackham/tabsets.wezterm) as shown below:

```lua
---@type Tabsets
local tabsets = wezterm.plugin.require("https://github.com/srackham/tabsets.wezterm")
```
