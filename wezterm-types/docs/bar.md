### `bar.wezterm`

[`adriankarlen/bar.wezterm`](https://github.com/adriankarlen/bar.wezterm) as shown below:

```lua
---@type BarWezterm
local bar_wezterm = wezterm.plugin.require("https://github.com/adriankarlen/bar.wezterm")
```
