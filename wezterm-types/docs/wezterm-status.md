### `wezterm-status`

You can import type annotations for
[`yriveiro/wezterm-status`](https://github.com/yriveiro/wezterm-status) as shown below:

```lua
---@type WeztermStatus
local status = wezterm.plugin.require("https://github.com/yriveiro/wezterm-status")
```
