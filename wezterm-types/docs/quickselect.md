### `quickselect.wezterm`

You can import type annotations for
[`quantonganh/quickselect.wezterm`](https://github.com/quantonganh/quickselect.wezterm) as shown below:

```lua
---@type Quick_Select
local quickselect = wezterm.plugin.require("https://github.com/quantonganh/quickselect.wezterm")
```
