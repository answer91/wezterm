### `wezterm-replay`

You can import type annotations for
[`btrachey/wezterm-replay`](https://github.com/btrachey/wezterm-replay) as shown below:

```lua
---@type Replay
local replay = wezterm.plugin.require("https://github.com/btrachey/wezterm-replay")
```
