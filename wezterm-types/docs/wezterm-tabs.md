### `wezterm-tabs`

You can import type annotations for
[`yriveiro/wezterm-tabs`](https://github.com/yriveiro/wezterm-tabs) as shown below:

```lua
---@type WeztermTabs
local tabs = wezterm.plugin.require("https://github.com/yriveiro/wezterm-tabs")
```
