### `wezterm-cmdpicker`

You can import type annotations for
[`abidibo/wezterm-cmdpicker`](https://github.com/abidibo/wezterm-cmdpicker) as shown below:

```lua
---@type CmdPicker
local picker = wezterm.plugin.require("https://github.com/abidibo/wezterm-cmdpicker")
```
