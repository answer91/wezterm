# WezTerm 快捷键配置文档

## 修饰键说明

| 平台 | 主修饰键 | 组合修饰键 |
|------|----------|------------|
| Linux | `ALT` | `ALT|CTRL` |

---

## 功能键

| 按键 | 功能 |
|------|------|
| `F1` | 进入复制模式 |
| `F2` | 打开命令面板 |
| `F3` | 显示启动器 |
| `F4` | 模糊搜索标签页 |
| `F5` | 模糊搜索工作区 |
| `F11` | 全屏切换 |
| `F12` | 显示调试覆盖层 |

---

## 基础操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` | 复制到剪贴板 |
| `Ctrl+Shift+V` | 从剪贴板粘贴 |

---

## 标签页操作

### 新建和关闭

| 快捷键 | 功能 |
|--------|------|
| `Alt+T` | 新建标签页 |
| `Alt+Ctrl+W` | 关闭当前标签页 |

### 导航

| 快捷键 | 功能 |
|--------|------|
| `Alt+[` | 切换到上一个标签页 |
| `Alt+]` | 切换到下一个标签页 |
| `Alt+Ctrl+[` | 向左移动标签页 |
| `Alt+Ctrl+]` | 向右移动标签页 |

### 数字切换

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+1` | 切换到第1个标签页 |
| `Ctrl+Shift+2` | 切换到第2个标签页 |
| `Ctrl+Shift+3` | 切换到第3个标签页 |
| `Ctrl+Shift+4` | 切换到第4个标签页 |
| `Ctrl+Shift+5` | 切换到第5个标签页 |
| `Ctrl+Shift+6` | 切换到第6个标签页 |
| `Ctrl+Shift+7` | 切换到第7个标签页 |
| `Ctrl+Shift+8` | 切换到第8个标签页 |
| `Ctrl+Shift+9` | 切换到最后一个标签页 |

## 窗口操作

| 快捷键 | 功能 |
|--------|------|
| `Alt+N` | 新建窗口 |
| `Alt+Ctrl+Enter` | 最大化窗口 |

---

## 窗格操作

### 分割

| 快捷键 | 功能 |
|--------|------|
| `Alt+\` | 垂直分割窗格 |
| `Alt+Ctrl+\` | 水平分割窗格 |

### 缩放和关闭

| 快捷键 | 功能 |
|--------|------|
| `Alt+Enter` | 切换窗格缩放状态 |
| `Alt+W` | 关闭当前窗格 |

### 导航（Vim 风格）

| 快捷键 | 功能 |
|--------|------|
| `Alt+Ctrl+H` | 激活左侧窗格 |
| `Alt+Ctrl+J` | 激活下方窗格 |
| `Alt+Ctrl+K` | 激活上方窗格 |
| `Alt+Ctrl+L` | 激活右侧窗格 |

### 快速选择和交换

| 快捷键 | 功能 |
|--------|------|
| `Alt+Ctrl+P` | 显示窗格选择器（数字0-9交换） |

### 滚动

| 快捷键 | 功能 |
|--------|------|
| `Alt+U` | 向上滚动5行 |
| `Alt+D` | 向下滚动5行 |
| `PageUp` | 向上滚动一页 |
| `PageDown` | 向下滚动一页 |

## Leader 模式

**前缀键**: `Alt+Ctrl+Space`

**超时**: 1000 毫秒

| 快捷键 | 功能 |
|--------|------|
| `Space + f` | 进入字体调整模式 |
| `Space + p` | 进入窗格调整模式 |

---

## 键表 (Key Tables)

### 字体调整模式 (resize_font)

进入方式: `Leader + f`

| 按键 | 功能 |
|------|------|
| `k` | 增大字体 |
| `j` | 减小字体 |
| `r` | 重置字体 |
| `Escape` / `q` | 退出模式 |

### 窗格调整模式 (resize_pane)

进入方式: `Leader + p`

| 按键 | 功能 |
|------|------|
| `k` | 向上调整窗格 |
| `j` | 向下调整窗格 |
| `h` | 向左调整窗格 |
| `l` | 向右调整窗格 |
| `Escape` / `q` | 退出模式 |

---
